आम्ही कोणती माहिती गोळा करतो?

स्वयंचलित माहिती (Anonymous)

  • Device ID (anonymous — नाव किंवा ओळख नाही)
  • App उघडण्याची वेळ व तारीख
  • कोणते Page/Feature वापरले
  • Search केलेले शब्द (location name फक्त)

तुम्ही स्वेच्छेने दिलेली माहिती

  • तक्रार/Grievance: नाव, मोबाईल, तक्रार विषय
  • GPS Location — फक्त तुमच्या परवानगीनंतर
  • फोटो — फक्त तक्रार फॉर्ममध्ये, ऐच्छिक
📍
GPS / Location माहिती

KumbhConnect तुमच्या परवानगीशिवाय GPS access करत नाही.

  • Location access करण्यापूर्वी स्पष्ट Consent popup दाखवले जाते
  • Location फक्त SOS/Grievance पाठवताना वापरली जाते
  • तुमचे Live Location कधीही स्वयंचलितपणे share होत नाही
  • GPS नाकारल्यास Manual location entry उपलब्ध आहे
💬
WhatsApp शेअरिंग

WhatsApp वर माहिती फक्त तुमच्या action नंतरच पाठवली जाते — कधीही automatically नाही.

  • तक्रार पाठवण्यापूर्वी Consent popup दाखवतो
  • "तुमची माहिती जवळच्या मदतीसाठी पाठवली जाईल" — हे नेहमी दाखवतो
  • SOS मध्ये Location WhatsApp वर जाते — फक्त तुमच्या संमतीनंतर

  • ❌ तुमचा डेटा कधीही विकत नाही
  • ❌ तृतीय पक्षाला (Third Party) माहिती देत नाही
  • ❌ Advertisements साठी डेटा वापरत नाही
  • ❌ तुमचे नाव किंवा मोबाईल नंबर स्टोअर करत नाही (analytics मध्ये)
  • ❌ Cookies track करत नाही

डेटा कुठे साठवला जातो?
  • LocalStorage: तुमच्याच device वर — anonymous analytics events
  • Google Sheets (Apps Script): Anonymous event counts — कोणतीही ओळख नाही
  • Firebase (भविष्यात): Anonymous usage data फक्त
  • तक्रारी/SOS: WhatsApp द्वारे थेट Police/Guide कडे — आम्ही स्टोअर करत नाही

Offline & PWA सुरक्षा

KumbhConnect एक Progressive Web App (PWA) आहे. Offline mode मध्ये:

  • Emergency numbers व verified locations cache मध्ये सुरक्षित
  • Network नसताना local data वापरतो
  • Service Worker द्वारे secure caching

What Information Do We Collect?

Automatic (Anonymous)

  • Anonymous Device ID (no name or identity)
  • App open time and date
  • Which pages/features were used
  • Search keywords (location names only)

Voluntarily Provided

  • Grievance/Complaint: Name, mobile number, complaint details
  • GPS Location — only after your explicit permission
  • Photos — optional, only in complaint form
📍
GPS / Location Data

KumbhConnect never accesses GPS without your permission.

  • A clear consent popup is shown before any location access
  • Location is used only when submitting SOS or Grievance
  • Your live location is never shared automatically
  • Manual location entry is available if GPS is denied
